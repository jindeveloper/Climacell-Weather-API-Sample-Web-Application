Weather Code Icons
========

#### Icons for ClimaCell API ####
Is it foggy? Rainy? A clear day? If you need a description of what the weather is like, simply request the weather code. As our beloved [ClimaCell API](https://developer.climacell.co/) consumers, we provide full access to our library of weather icons. 

### What's Includes ###
SVG icons, colored and black, of all available weather codes: `freezing_rain_heavy`, `freezing_rain`, `freezing_rain_light`, `freezing_drizzle`, `ice_pellets_heavy`, `ice_pellets`, `ice_pellets_light`, `snow_heavy`, `snow`, `snow_light`, `flurries`, `tstorm`, `rain_heavy`, `rain`, `rain_light`, `drizzle`, `fog_light`, `fog`, `cloudy`, `mostly_cloudy`, `partly_cloudy`, `mostly_clear`, `clear`

![Weather Code Icons](./PREVIEW.svg)

### Usage ###
Be sure to place a “Powered by ClimaCell” attribution in your application if you use our icons.
